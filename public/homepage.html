Avi-Sustain Index

Commercial aviation plays a major role in global carbon emissions, yet passengers rarely see how individual flights compare in environmental impact.

The Avi-Sustain Index is a transparency tool that helps users estimate the carbon emissions of a flight route using real airport location data and distance-based calculations.

Instead of complex aviation datasets, this project presents emissions in a clear, comparable metric: kilograms of CO2 per passenger.

What the tool provides

  • Estimated flight distance using real airport coordinates
  • Carbon emissions per passenger
  • Visual comparison of recent searches
  • Interactive route map
